# 介绍

此脚本的作用是一直ping某个ip,并记录log.

# 部署方法

用编辑器打开**NetMonitor.bat**在其中第**6**行修改目标IP,第**7**行可修改ping间隔,默认是3秒ping一次.
然后运行**点这运行.vbs**,用这个vbs文件运行**不会显示任何东西**,不要直接点bat文件,这样会有命令行窗口.

# 注意要点
